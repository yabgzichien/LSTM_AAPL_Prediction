🎯 To Improve Accuracy
1. Attention Mechanism (Biggest Impact)
Add a self-attention layer after the LSTM/GRU — this lets the model learn which timesteps in the lookback window matter most. It's the single best architectural upgrade for sequence models.

2. Ensemble the Two Models
Average the LSTM + GRU predictions. Simple ensembling often beats either model alone:

python
ensemble_pred = (pred_lstm + pred_gru) / 2
3. Class Weight Balancing
Stock returns are rarely 50/50 up/down. If imbalanced, the model biases toward the majority class. Use class_weight in .fit() to penalize the minority class more.

4. Expand Lookback Search
You're only testing [30, 60]. Try [10, 20, 30, 45, 60, 90] — shorter windows sometimes capture momentum better.

5. Add Sentiment / External Signal
Fear & Greed Index or VIX derivatives (you already have VIX, but VIX term structure slope is more predictive)
Lagged features — add Return_lag1, Return_lag5 to give the model explicit momentum memory
